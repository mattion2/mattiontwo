"Fungsi Kuadrat"

Bentuk dari Fungsi Kuadrat : y = ax² + bx + c, dengan a ≠ 0

Grafik dari fungsi kuadrat menyerupai "Parabola", sehingga dapat dikatakan juga sebagai fungsi parabola.

A. Grafik fungsi kuadrat jika bentuknya y=ax²

Siswa diminta untuk menggambar grafik fungsi y=x², y=-x², y=2x² Setelah itu siswa menyimpulkan apa perbedaan dari grafik-grafik fungsi tersebut.

B. Grafik fungsi kuadrat jika bentuknya y=ax² + bx + c

Siswa diminta untuk menggambar grafik y = x² - 3x + 2, y = x² + 2x, y = -x² -5x -4. Setelah mengamati gambar grafik fungsi yang telah dibuat, siswa dapat menyimpulkan beberapa Sifat Fungsi Kuadrat yaitu :

1. Nilai b pada grafik y = ax² + bx + c menunjukkan letak koordinat titik puncak dan sumbu simetri .

2. Sumbu Simetri yaitu x = -b/2a.

3. Jika a > 0, grafik y = ax²+ bx + c memiliki titik puncak minimum yaitu -D/4a.

4. Jika a < 0, grafik y = ax² + bx + c memiliki titik puncak maksimum yaitu -D/4a.

5. Nilai c pada grafik y = ax²+ bx + c menunjukkan titik perpotongan grafik fungsi kuadrat tersebut dengan sumbu-y, yakni pada koordinat (0, c).

C. Diskriminan

Diskriminan yaitu suatu nilai pada persamaan (umumnya persamaan kuadrat) yang membedakan banyaknya akar persamaan itu sendiri.

Diskriminan juga memiliki arti hubungan antara koefisien dalam persamaan kuadrat untuk mencari akar persamaan dan ciri – ciri yang lainnya.

Rumus diskriminan : D= b² - 4ac

Diskriminan

Grafik akan memotong sumbu x di dua titik jika D > 0 (positif)

Grafik akan memotong sumbu x di satu titik (menyinggung sumbu x) jika D = 0

Grafik tidak memotong sumbu x jika D < 0 (negatif).

Dari penjelasan diatas, dapat diambil beberapa kesimpulan penting yaitu :

1. Nilai a pada fungsi y = ax²+ bx + c akan mempengaruhi bentuk grafiknya.

*Jika a positif maka grafiknya akan terbuka ke atas.

*Jika a negatif maka grafiknya akan terbuka ke bawah.

2. Jika nilai a semakin besar maka grafiknya menjadi lebih “kurus”.

3. Jika a>0 maka terdapat tiga kemungkinan D yaitu seperti pada gambar diatas. Untuk a<0 dapat dicoba sendiri.
