1. Nilai minimum dari fungsi kuadrat f(x) = (x-2a)x² + 3b adalah 21 dan grafik fungsinya berpotongan dengan sumbu -y di titik dengan ordinat 25. Nilai dari a² + b adalah ....

Pembahasan
Diketahui : Nilai minimum = 21
f(x) = (x-2a)² + 3b
f(x) = ( x² - 4ax + ( 4a² + 3b)
-D/4a = (b² - 4.a.c)/4.a
21= -((-4a)² -4.1.(4a² + 3b))/4.1
21= 12b / 4
b = 7
perpotongan dengan sumbu y, (x=0) adalah 25
25 = x² - 4ax + ( 4a² + 21)
25 = 4a² + 21
4 = 4a²
1 = a
Jadi, a² b = 1 + 7 = 8

3. Batas p jika f(x) = (p+1)x² - 2px + p-4 memotong sumbu x di satu titik adalah ..

Pembahasan
f(x) = (p + 1)x² – 2px + p – 4
a = (p + 1); b = -2p; c = p – 4
D = (-2p)² – 4. (p + 1). (p – 4)
= 12p + 16
syarat memotong di satu titik D=0
12p + 16 = 0
12p = -16
p = -4/3

6. Jika f adalah fungsi kuadrat yang grafiknya melalui tiitk (1,0),(4,0), dan (0,-4), maka nilai dari f(5) adalah..

Pembahasan
Titik yang dilalui fungsi f kebetulan merupakan titik potong grafik terhadap sumbu-X, yaitu (1,0) dan (4,0)sehingga rumus fungsinya adalah :
y = a(x-1)(x-4)
Grafik fungsi kuadrat melalui titik (0,-4) berarti
y = a(x-1)(x-4)
-4 = a(0-1)(0-4)
-4 = a(-1)(-4)
a = -1
Rumus fungsi kuadrat adalah
f(x) y = -1 (x-1)(x-4)
f(x) = -1 (x² - 5x + 4)
= -x² + 5x - 4
Jadi nilai dari f(5) adalah :
-25 + 25 -4 = -4

8. Jika titik P(-3,5) dan Q(7,5) terletak pada grafik fungsi f(x) = p(x-q)² + r, maka q adalah ...

Pembahasan
Karena titik P(-3,5) terletak pada grafik fungsi f(x) = y, maka subtitusikan x=-3 dan y= 5, sehingga diperoleh :
5 = p(-3-q)² + r ..(1)
Begitu juga dengan titik Q(7,5)
Subtitusikan x=7 dan y=5 sehingga diperoleh :
5 = p(7-q) + r ..(2)
Eliminasi r dari kedua persamaan di atas , sehingga di dapat :
0 = p(-3-q)² - p(7-q)²
0 = (-3-q)² - (7-q)²
0 = (-3-q+7-q)(-3-q-7+q)
Sehingga didapatkan q=2
